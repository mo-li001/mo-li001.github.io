BMC、BIOS简单介绍-BIOS

BIOS

BIOS主要功能是上电自检，硬件初始化，包括 CPU 内存 PCIE等等

同时现代BIOS提供图形化界面给用户进行硬件配置，满足多样化配置需求

BMC、BIOS简单介绍-BMC

BMC

服务器与PC的最大差别就是服务器有BMC而PC没有

BMC在服务器上有专门的芯片，独立于OS，在接上电源时就开始工作，可以提供提供故障检测 固件升级 远程控制等等功能

BIOS常用选项配置

参考机型：Inspur SA5212M5

BIOS常见设置 位置 作用&影响简述
启动项 Save & Exit → Boot override 查看启动项，手动选择启动项启动
启动模式UEFI/LEGACY Boot 查看修改启动模式
CSM Advanced → CSM configuration 查看修改各部件启动模式
VT Processor → IIO Configuration 查看修改虚拟化技术
HT Processor → Processor Configuration 查看修改超线程技术
Network Stack Advanced → Network Stack 查看修改UEFI下PXE功能
Onboard Lan Control Advanced → Onboard Lan Control 查看修改板载网口开关
VMD Processor → IIO Configuration → VMD 查看修改VMD配置
BIOS常见硬件配置查看 位置 作用&影响简述
CPU信息 Processor → Processor Information 查看CPU信息
硬盘信息 Chipset → PCH ssata Configruation 查看硬盘信息
内存信息 Processor → Memory Topology 查看内存信息
网卡信息 Advanced → 实际网卡项 查看网卡信息
BIOS内BMC设置查看修改 位置 简述
fru信息查看 Server Mgmt → View FRU information 查看FRU
BMC用户名密码修改 Server Mgmt → BMC User Settings 增加/修改/删除BMC用户及密码
BMC IP模式修改 Server Mgmt → BMC network configuration 查看修改BMC IP模式

设置修改完毕后：

1、Save & Exit，代表保存退出

2、Exc，代表不保存退出

BMC web页面常用功能 作用简述（补充）
fru信息查看 fru上存放的关键字段如Product Serial与服务器SN保持一致
BMC主机名查看 BMC主机名需要与服务器SN保持一致
传感器信息查看 查询服务器相关传感器状态，如CPU；内存在位信息、出入风口温度、风扇转速等等
事件日志查看 查询服务器的硬件事件日志，明显的硬件故障可以此处看到
故障日志收集 收集服务器硬件的事件日志，用于分析硬件故障
RAID界面信息查看-菜单栏 该界面包含内容简介 关键状态判断
VD mgmt 查看/修改阵列卡，硬盘组和虚拟组的配置 Raid level，raid组有多少硬盘
PD mamt 查看/修改物理硬盘的配置 硬盘状态
Crtl mgmt 查看/修改阵列卡的属性 Boot device
Properties 查看阵列卡的信息 阵列卡状态，SN，FW版本

常见的Linux服务&配置文件

常见的服务&配置文件 服务&配置文件简述 关键文件、命令说明
network服务 只有配置了IP，启用了网络服务，服务器才可以访问网络 ifcfg-eth1、ifcfg-eth0：存放IP配置信息service network start/restart，启动/停止/重启网络服务
ssh服务 只有ssh服务正确配置，并启用后，服务器才可以通过IP地址远程ssh登录 sshd_config、sshd_config.l：存放ssh配置文件service sshd start/stop/restart，启动/停止/重启ssh服务
70文件 70文件规定了网卡与名字绑定规则 /etc/udev/rules.d/70-persistent-net.rules修改70文件后需要重启生效
fstab文件 fstab文件，配置了开机自动挂载的分区，在上面配置的挂载条目在OS启动时会自动进行挂载 位置/etc/fstab
